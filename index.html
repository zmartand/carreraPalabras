Bienvenido al juego de Carrera de Palabras

Instrucciones

Objetivo

Introduce la palabra que ves en la pantalla de manera correcta y esta se enviará automáticamente despues de escribirla

¿Cómo jugar?

Ingresa la palabra mostrada en la pantalla dentro del campo de texto (puedes usar mayúsculas o minúsculas)

Tiempo

Puedes elegir el tiempo inicial (el tiempo es limitado para cada palabra). Al pasar de nivel el contador disminuirá 1 segundo.

Vidas

Puedes elegir el número de vidas iniciales, por cada error y límite de tiempo agotado pierdes una vida

Niveles

El juego se hace mas desafiante progresivamente. Pasas de nivel cada 5 palabras correctas y con cada nivel tendrás menos tiempo para escribir las palabras

Fin de juego

El juego termina cuando se te acaben las vidas
